# red-area

### 题目

红色区域的大小是多少？

```
.box { 
    width: 200px; 
    height: 200px; 
    padding: 20px; 
    margin: 20px; 
    background: red; 
    border: 20px solid black; 
    box-sizing: border-box; 
} 
```



### 解题

#### 思路

* 当box-sizing的属性为border-box时，宽包含了border和padding，而红色区域包含padding不包含border，所以红色区域为 (200 - 40) * (200 - 40) = 160*160
