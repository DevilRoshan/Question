### 解题

#### 思路

就是一个sort排序的书写

#### 代码

```javascript
/**
 * 根据传入参数n(数字)对一维数组(纯数字)按照距离n最近的顺序排序
 * 距离即数字与n的差值的绝对值
 */
const arr = [7, 28, -1, 0, 7, 33]
function sort(n) {
	return arr.sort((a, b) => {
    return Math.abs(n - a) - Math.abs(n - b)
  })
}
```
